# 🏠 Interior Design 

## 📌 Project Title

Modern Interior Design & Furniture E-Commerce Frontend Website

---

## 📖 Brief One Line Summary

A modern Interior Design frontend web application with furniture e-commerce features, project portfolios, order management, wishlist, cart system, and email integration built using React.js.

---

## 🧾 Overview

This project is a frontend-based Interior Design and Furniture E-Commerce website developed using modern web technologies like HTML, CSS, JavaScript, and React.js.

The application allows users to explore interior design ideas, view different portfolio categories, purchase furniture products, manage orders, and connect with the company through integrated forms and email services.

The website includes multiple interactive features such as wishlist management, cart functionality, checkout system, discount code support, dummy payment integration, order tracking, and portfolio showcase pages.

Users can also upload inspiration images and fill out forms to receive quick interior design ideas and connect directly with the company.

The project mainly focuses on creating a clean UI/UX experience, reusable component structure, and modern frontend architecture.

---

## ❗ Problem Statement

Many interior design websites only provide static information and lack interactive user features such as personalized design inspiration, furniture purchasing, order management, and smooth user experience.

This project solves these problems by providing:

* A modern responsive frontend interface
* Interactive e-commerce functionality
* Interior inspiration and portfolio showcase
* User-friendly checkout and order management system
* Email-based communication between users and admin
